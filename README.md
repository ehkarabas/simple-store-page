## Description

Design of a fully responsive website using Flexbox layout and featuring anchor links, fixed & sticky bars.

## Goals

Practicing on;

* Flexbox
* Position properties
* CSS Media Queries
* Animation with pseudo elements

```
    |-- style.css
```


## Methodology

* Used

	* HTML elements
	* CSS selectors
	* CSS style attributes
	* Box model
	* Flexbox
	* Pseudo Elements
	* Position properties
	* External CSS files
	* CSS Media Queries


* Not used

	* Float, grid layouts
	* Display properties(flex excl.)
	* Bootstrap
	* JavaScript
		* and so on...
